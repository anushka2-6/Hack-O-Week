# ✈️ AI Travel Planner

## 📌 Overview

This project is an AI-based travel planner that helps users plan trips by suggesting destinations, travel details, and basic guidance based on user input. It simplifies travel planning using rule-based logic and user preferences.

---

## ⚙️ Features

* ✅ User input for travel preferences
* ✅ Suggests destinations and plans
* ✅ Basic rule-based decision making
* ✅ Easy-to-use command-line interface

---

## 🛠️ Technologies Used

* Python
* Basic AI logic (rule-based system)

---

## ▶️ How to Run

1. Make sure Python is installed
2. Run the script:

   ```bash
   python travel_planner.py
   ```

---

## 💬 Example

**Input:**
`I want to travel to a cold place`

**Output:**
`You can visit Manali or Shimla for a cold climate experience.`

---

## 📂 Project Purpose

This project is developed as part of **Hack-O-Week** to demonstrate how AI can assist in travel planning and recommendations.
