# 🤖 Student Query Preprocessing Chatbot

## 📌 Overview

This project is a simple rule-based chatbot that preprocesses user queries and provides responses based on keywords. It demonstrates basic Natural Language Processing (NLP) techniques without using heavy libraries like NLTK.

---

## ⚙️ Features

* ✅ Lowercasing text
* ✅ Punctuation removal
* ✅ Tokenization using regex
* ✅ Stopword removal (manual list)
* ✅ Spelling correction using TextBlob
* ✅ Rule-based chatbot responses

---

## 🛠️ Technologies Used

* Python
* `re` (Regular Expressions)
* `string` module
* TextBlob

---

## ▶️ How to Run

1. Install TextBlob:

   ```bash
   pip install textblob
   ```
2. Run the script:

   ```bash
   python chatbot.py
   ```

---

## 💬 Example

**Input:**
`What is machine learning?`

**Output:**
`Machine learning is a branch of Artificial Intelligence that enables systems to learn from data.`

This project is created as part of **Hack-O-Week** to demonstrate preprocessing of student queries and building a basic FAQ chatbot.
